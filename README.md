# ScapeProtocol
## Premissa do game:<br>
Após um experimento biológico sair do controle, um laboratório subterrâneo entra em lockdown total e libera criaturas mutantes pelos corredores. Dois cientistas sobreviventes ficam presos dentro da instalação e precisam trabalhar juntos para atravessar diferentes setores do laboratório, resolver sistemas de segurança e encontrar uma saída antes que o protocolo de autodestruição seja ativado.

## Level design:<br>
- Zona A: Entrada e Bloqueio (Lado Esquerdo)Objetivo: Os dois jogadores começam aqui. A porta principal ("LV.3") está trancada.Mecânica: O jogador que hackeia sistemas precisa interagir com o terminal de controle no canto superior esquerdo.Cooperação: Enquanto um hackeia, o outro jogador deve se posicionar perto do tanque central para observar se criaturas mutantes  aparecem pelos dutos.
- Zona B: O Corredor de Energia (Centro)Objetivo: Atravessar para a ala leste.Obstáculo: Os postes elétricos (bobinas) que soltam faíscas na imagem funcionam como perigo ambiental.Mecânica: Os jogadores precisam sincronizar o movimento para passar entre as descargas elétricas.Ação: Um jogador pode encontrar uma alavanca  que desliga temporariamente os postes para o parceiro passar.
- Zona C: Laboratório de Experimentos (Lado Direito)Objetivo: Coletar uma bateria ou chave para a saída final.Mecânica: Interagir com os tanques de experimento e a bancada de químicos.Desafio: Ativar o mecanismo físico que libera o item, o que pode disparar o robô de segurança  localizado no canto inferior direito.
