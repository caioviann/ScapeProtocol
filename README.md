# ScapeProtocol
## Premissa do game:<br>
Após um experimento biológico sair do controle, um laboratório subterrâneo entra em lockdown total e libera criaturas mutantes pelos corredores. Dois cientistas sobreviventes ficam presos dentro da instalação e precisam trabalhar juntos para atravessar diferentes setores do laboratório, resolver sistemas de segurança e encontrar uma saída antes que o protocolo de autodestruição seja ativado.
